Tim Berners Lee & The World Wide Web

The World Wide Web was created by Tim Berners Lee, a British computer scientists working at a physics lab CERN in 1989. The first website went live on August 6, 1991. He wrote the first web client and server in 1990. Initially, Berners-Lee’s innovation was intended to help scientists and researchers share data across the internet. The lack of resources forced Berners-Lee to think outside the box and create standards for using the internet:
  • HTTP
  • HTML
  • URL

HTTP

HTTP, known as the HyperText Tranfer Protocol, is used by web browsers to request pages and by web servers to share them over the internet. This provolone ran over TCP/IP. It is the foundation of any data exchange on the internet and it is a client-server protocol, where requests are started by the web browser.

HTML

HTML, known as the HyperText Markup Language, was created as a set of instructions to display web pages on the screen. This markup tells web browsers how to display a web page’s words and images. HTML allows authors to publish their documents with different headings, tables, fonts, lists, photos, etc. HTML text is saved as an html file and viewed through a Browder like Internet Explorer, or Google Chrome.

URL

URL,known as Uniform Resource Locator, which is a reference to an ip address on the internet. It was built on the existing Domain Name System, adding prefixes such as nttp:// , ftp://, telnet:// , http:// to access newsgroups, remote logins, and web pages.

Berners-Lee's Impact

Berners- Lee’s greatest contribution was his simple method for an author to link a page to documents and users clicking the link to follow it anywhere on the internet. The Web functioned as a catalog for resources on the internet. Browser programs were used to access those resources. He changed the world by inventing the World Wide Web. However, he knew that his invention had to be protected, so he created World Wide Web Foundation to defend the open web for the public good.
